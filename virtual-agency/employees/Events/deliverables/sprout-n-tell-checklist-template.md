# Sprout N Tell — Event Checklist Template

> The canonical prep-and-production checklist for a Sprout N Tell / Show n Tell showcase.
> Built from the real checklists across the events: Show n Tell (5/19), Sprout n Tell (6/26), Vol. 3 (7/24).
> **This doc is the source of truth.** It is mirrored in the app as the `EVENT_CHECKLIST_TEMPLATES["Sprout N Tell"]` constant in `components/CRMManager.jsx`; when this list changes, update that constant so the in-app **Import checklist template** button stays in sync.

---

## How to use it

**In the app:** open an event → **Add / Edit → Checklist tab → 📋 Import a checklist template → ＋ Import "Sprout N Tell" checklist.** Set the event date on the Overview tab first. Each item's due date is calculated from the event date using the offsets below, and the dated items then populate the event's calendar automatically. Re-importing skips items already on the list.

**Reading the offsets:** `offset` = days relative to the event date. Negative = before the event, `0` = event day, positive = after.

## Notes

- Offsets are lead times, not hard rules. A show with a bigger lineup pulls the confirmation items earlier; a smaller one can compress the back half. Adjust dates on the calendar after import.
- Lineup items (Confirm Musicians / Confirm Artists / Artists-Music Confirmed) overlap on purpose: the first two are the outreach, the third is the lock. Keep all three or collapse to one per how the booking ran.
- The **THANK YOU FOLLOW UP** item at +7 is the handoff trigger: that is where the Communications Manager picks up the recap for the newsletter and the CRM Manager logs attendee touchpoints.
