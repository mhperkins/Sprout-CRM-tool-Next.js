# Sprout Society CRM

A relationship management tool built for the non-profit Sprout Society.

## Features
- Contact management with relationship status and health tracking
- Event tracking with linked contacts and task management
- Outreach log for communication history
- Claude integration to generate and import contact profiles via JSON

## Stack
Next.js, Supabase, Claude API, Tailwind CSS, Vercel

## Status
Live and actively used in production
