Newsletter Tutorial — for non-techy users

Docs only Effort: medium 10 steps · 10 screenshots No app code change
🌱

A plain-English HTML walkthrough of the newsletter feature for people who have never seen the app. Real annotated screenshots captured from the running app — no jargon, just click-by-click. Includes a portable single-file version with every image baked in, so it opens or emails anywhere.

What it walks through

  • Pick a template — Monthly Roundup vs Quick Hit
  • The editor — worksheet left, live preview right
  • Write sections — the ✨ Polish & 🖼 Add image buttons
  • Test on yourself first, then approve & send to a list
  • Find it again under Saved newsletters

How it was built

  • Drove the live dev server with Playwright to capture 10 hi-res shots
  • Numbered pins (percentage-positioned, so they scale) + plain legends
  • Sprout palette — ink / acid / banana / fuchsia / cyan, Lato
  • Inliner baked images to base64 → 0.95 MB portable file
  • Both throwaway scripts deleted after
