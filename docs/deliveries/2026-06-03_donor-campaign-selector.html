Donor Campaign Selector 🌱
Sprout CRM Next · synced from Givebutter
Shipped
Donors can now be tagged to a Givebutter campaign. The field appears only when a contact's Bucket = Donor, in both the Add and Edit modals, as a searchable dropdown synced from your live campaign list.
🎯
Two new fields
campaign (title, shown) + a hidden campaign_id (stable Givebutter id) on every contact record.
🛡️
Rename-proof by design
campaign is a free string, not an enum — a renamed Givebutter campaign never fails validation and silently drops the contact.
🔁
Synced static list
The dropdown lives in CAMPAIGN_OPTS, refreshed on request. The browser app can't reach the MCP, so no API key in the app.
🔑
Future-ready key
The hidden id is the stable key for a future donor→campaign auto-sync against actual Givebutter giving.
