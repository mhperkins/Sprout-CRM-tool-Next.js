Editable checklist calendar
on the event detail page

App code · Effort: medium

The event checklist calendar was read-only beyond a mark-complete toggle. It's now a full planning surface — edit, add, and re-schedule items right on the month grid.

✏️ Edit name + date

Click an item → popover with an editable name field, a due-date picker, complete toggle, and delete.

➕ Click a day to add

Click any day cell → inline input prefilled with that date. Enter to add. Works on an empty checklist too.

✋ Drag to move

Drag an item dot to another day to re-date it. Days outline dashed-cyan while dragging.

🗓️ No-date section

Inline-editable rows; drag a dated item onto it to clear its date.

Bug caught + fixed same session: first build threw item is not defined — the edit popover was rendered outside the items.map. Moved it inside the loop, each dot wrapped in its own relative container. Build passes.
