A standing rule: the word effort (or judgment on any big task) triggers an upfront effort tier before work begins. Debugging splits the dial into diagnose / fix.

⚙️ How it fires

  • Open with Effort: <level> — <why>
  • Debug → diagnose / fix, fix never exceeds diagnosis
  • Data-loss surfaces → max on both
  • Re-tier mid-task autonomously, never stop to ask

🎚️ The dial — match effort to the cost of being wrong

LowKnown pattern, low blast radius. Solo + quick check.
MediumTouches a pipeline/store. Solo + live-verify, review medium.
HighSilent/behavioral correctness. Design-first + adversarial verify.
Max/UltraCross-cutting or data-integrity. Reserve for these.
