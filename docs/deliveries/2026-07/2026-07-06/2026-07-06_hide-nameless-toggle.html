Hide nameless contacts by default

Effort: Low main c448fb0 1 file · CRMManager.jsx

The contact list now hides email-only (nameless) contacts by default — the inverse of the existing “Needs a name” filter. Toggle it off anytime to bring them back.

What shipped

🙈 New toggle
“Hide nameless” button on the filter bar, default ON (hideNameless = true).
🔀 Mutually exclusive
Turning on either “Hide nameless” or “Needs a name” switches the other off — they’re opposites.
🧹 Clear resets to default
✕ Clear restores the default view (nameless hidden) and appears whenever it’s toggled off.
🏷️ Badge stays honest
The “filtered” chip ignores hide-nameless, since it’s the default state — not an active filter.
